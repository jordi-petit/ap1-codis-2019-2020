## Operacions sobre els enters en Python (`int`)

| Operador | Operació |
|----------|----------|
| `+`      | suma     |
| `-`      | resta     |
| `*`      | producte     |
| `//`      | divisió entera (:warning:) |
| `%`      | rest de la divisió entera  |
| `**`      | exponenciació (:smiley:)  |

## Comparació entre els enters

| Operador | Operació |
|----------|----------|
| `<`      | més petit     |
| `>`      | més gran     |
| `<=`      | més petit o igual |
| `>=`      | més gran o igual |
| `==`      | igual  |
| `!=`      | diferent |

## Nota

En Python sí que es poden escriure expressions com `a <= b <= c`.
